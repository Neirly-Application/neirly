API disponibili

Le API sono accessibili tramite chiamate REST e supportano formati JSON e XML.

GET /api/v1/resources
Host: api.example.com
Accept: application/json

Come iniziare

Segui questi semplici passi per iniziare:

  1. Registrati sul nostro portale.
  2. Ottieni una chiave API personale.
  3. Consulta la sezione Endpoint API per dettagli tecnici.

Endpoint API

Metodo URL Descrizione
GET /api/v1/users Recupera la lista utenti
POST /api/v1/users Crea un nuovo utente
DELETE /api/v1/users/{id} Elimina un utente
